# AKShare 开户数据获取模块

## 简介

本模块提供基于 AKShare 的投资者开户数据获取功能，数据来源为中国结算（中国证券登记结算有限公司）月度统计数据。

## 数据说明

- **数据源**: 东方财富网 / 中国结算
- **接口**: `stock_account_statistics_em()`
- **更新频率**: 月度（每月中旬公布上月数据）
- **历史数据**: 支持一次性获取完整历史数据
- **数据延迟**: AKShare 通常在官方公布后 1-2 天内同步
- **⚠️ 重要提示**: 经测试，该接口目前最新数据截至 **2023年8月**，后续数据暂未更新

## 主要字段

| 字段名 | 说明 | 单位 |
|--------|------|------|
| 数据日期 | 统计月份 | YYYY-MM |
| 新增投资者-数量 | 当月新增投资者数量 | 万户 |
| 新增投资者-环比 | 新增数量环比增长率 | % |
| 新增投资者-同比 | 新增数量同比增长率 | % |
| 期末投资者-总量 | 累计投资者总数 | 万户 |
| 期末投资者-A股账户 | A股账户数量 | 万户 |
| 期末投资者-B股账户 | B股账户数量 | 万户 |
| 沪深总市值 | 沪深两市总市值 | 亿元 |
| 沪深户均市值 | 户均市值 | 万元 |
| 上证指数-收盘 | 该月最后一个交易日收盘价 | 点 |
| 上证指数-涨跌幅 | 该月上证指数涨跌幅 | % |

## 快速开始

```python
import akshare as ak

# 获取全部历史数据
df = ak.stock_account_statistics_em()
print(df.tail())

# 获取最新一条记录
latest = df.sort_values('数据日期').iloc[-1]
print(f"最新数据: {latest['数据日期']}, 新增开户: {latest['新增投资者-数量']} 万户")
```

## 应用场景

1. **市场情绪分析**: 新增开户数反映散户入市热情
2. **顶底判断**: 开户数激增往往出现在市场高点
3. **长期趋势**: 期末投资者总量反映市场参与度
4. **配合行情**: 结合上证指数判断市场阶段

## 注意事项

1. 数据为月度统计，更新频率较低
2. 新增开户数存在季节性波动
3. 需结合其他指标综合判断市场
4. 数据更新时间为每月中旬
